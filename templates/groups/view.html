Group Help & Guide
What is a Group?

A group allows you to collaborate with team members on shared inventory lists. All members can view and manage lists together.

Managing Lists

Create new lists within this group using the Create New List button. All group members will have access to these lists.

Adding Members

Go to the Members tab and use Add Member to invite users to your group. They'll receive an invitation and can start collaborating immediately.

Group Settings

Group owners can edit settings, manage membership permissions, and control who can create or edit lists.

Deleting a Group

Only group owners can delete a group. This will permanently delete all lists and items in the group. Find the delete option in the Settings tab under "Danger Zone".
