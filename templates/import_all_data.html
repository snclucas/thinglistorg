Import Your Data

Restore your complete account data from an export file

ℹ️ About Full Data Import
This will import your complete account data including groups, lists, items, and all associated information. You can choose how to handle conflicts with existing data.

1. Select Your Export File

📄 Must be a .json file exported from "Export All Data"

2. Choose How to Handle Conflicts

If you have items that already exist with matching unique IDs, choose what happens:

Keep your existing groups. Only import new groups.

Replace your existing groups with imported data.

Keep existing groups, merge in imported data.

Keep your existing lists. Only import new lists.

Replace your existing lists with imported data (items not affected).

Keep existing lists and create new copies of imported lists.

Keep your existing items. Only import new items.

Replace your existing items with imported data.

💡 How does this work?
  • Unique IDs: Items are matched by their unique ID from export. If an item doesn't exist, it's imported as new.
  • Safe: Your existing data is never deleted. Choose "Skip" to be conservative.
  • Linked Data: Groups and lists are re-linked properly during import.
