Lists Help & Guide
What is a List?

A list is a collection of items you want to track. You can organize items by type, location, quantity, and more. Create lists for different categories like kitchen supplies, tools, inventory, etc.

Creating a New List

Click the Create New List button to get started. Give your list a name, optional description, and set visibility preferences.

Privacy Settings

Private: Only you can access. Public: Searchable and shareable. Hidden: Not searchable but accessible via direct link.

Using Tags

Add tags when creating a list to organize and categorize your lists. Use comma-separated values like "kitchen, urgent, supplies".

Sharing Lists

Share lists with group members by creating them within a group, or share individual lists using the sharing features in the list view.

Editing & Managing

Click on a list to view its items. From there, you can add items, edit details, and manage the list. Items can include quantities, locations, categories, and custom fields.
